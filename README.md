# 🚀 NOKIA SPACE SHOOTER (มหาโหด 3 วิ)

ยินดีต้อนรับสู่เกมยิงยานอวกาศสไตล์ Retro ยุค Nokia ครองเมือง! บังคับยานรบของคุณหลบกระสุนศัตรู และเก็บไอเทมเพื่อปลดปล่อยพลังทำลายล้างที่ข้าศึกต้องหลั่งน้ำตา!

---

## 🎮 วิธีการเล่น (Controls)
* **ลูกศร ขึ้น / ลง / ซ้าย / ขวา** : บังคับทิศทางยานรบ
* **Spacebar** : ยิงกระสุนปืนกล (กดรัว ๆ ได้เลย!)
* **ปุ่ม R** : รีสตาร์ทเริ่มเกมใหม่ (หลังจาก Game Over)

---

## 🔥 ระบบไอเทมพิเศษ (Power-Up)
ทุก ๆ **3 วินาที** จะมีกล่องไอเทมปริศนาลอยมาจากขอบจอขวา!
* **กล่องสี่เหลี่ยมสีดำ อักษร [ P ] สีเขียว**
* **ความสามารถ:** อัปเกรดปืนปกติให้กลายเป็น **"ปืนกล 3 แฉก (Triple Shot)"** ยิงกระจายทะลวงทุกกองทัพ จนต้องร้องขอชีวิต อุ๊แว้ๆๆๆ!

---

## 🏆 วิธีการนับคะแนน
* ทำลายยานศัตรูได้: `+100 คะแนน`
* เก็บกล่องไอเทมปืนได้: `+500 คะแนน`
